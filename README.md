# multi-page-website-cssday3

# TechTide Summer Internship 2026 – Day 3 CSS Flexbox Assignment

## Project Title

**TechTide Multi-Page Company Website**

---

## Objective

The objective of this project is to enhance a multi-page company website using **CSS Flexbox**. The project demonstrates how modern Flexbox layouts can be used to create responsive, organized, and professional web pages without using floats or CSS Grid.

---

## Technologies Used

* HTML5
* CSS3
* CSS Flexbox

---

## Flexbox Concepts Used

The following Flexbox properties were implemented throughout the project:

* `display: flex`
* `flex-direction`
* `justify-content`
* `align-items`
* `gap`
* `flex-wrap`
* `flex-grow`
* `flex-shrink`
